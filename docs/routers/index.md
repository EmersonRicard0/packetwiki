# Routers

Welcome to the PacketWiki routers section. Here you'll find configuration guides, command references, and troubleshooting tips for routers from all major vendors.

## Vendors

| Vendor | Models Covered | CLI Type |
|--------|---------------|----------|
| [Cisco](/routers/cisco/) | ISR 4000, ASR 1000, Catalyst 8000 | IOS / IOS-XE |
| [MikroTik](/routers/mikrotik/) | hEX, CCR, RB4011 | RouterOS (CLI / Winbox) |
| [Juniper](/routers/juniper/) | SRX, MX Series | Junos |
| [Ubiquiti](/routers/ubiquiti/) | UniFi Dream Machine, EdgeRouter | EdgeOS / UniFi |
| [Huawei](/routers/huawei/) | AR Series, NE Series | VRP |

## How to use this section

Each vendor page contains:

- **Overview** — general architecture, key features
- **Initial setup** — factory reset, first access, basic config
- **Configuration examples** — real CLI/GUI examples with explanations
- **Common issues** — troubleshooting known problems
- **Command reference** — quick reference table
