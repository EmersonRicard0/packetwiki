# Servidores Linux

Guias de configuração para servidores Linux voltados para infraestrutura de redes — serviços, firewall, monitoramento e mais.

## Distribuições

| Distro | Uso comum |
|--------|-----------|
| Ubuntu Server | Ambientes gerais, fácil adoção |
| Debian | Servidores estáveis de longa duração |
| Rocky Linux | Substituto RHEL para produção |
| AlmaLinux | Alternativa RHEL com suporte comercial |
| CentOS Stream | Preview do RHEL upstream |

## Tópicos disponíveis

### Serviços de Rede
- DHCP (isc-dhcp / Kea)
- DNS (BIND9 / Unbound)
- NTP (chrony / ntpd)
- VPN com WireGuard
- VPN com OpenVPN
- Proxy com Squid

### Firewall
- iptables
- nftables
- UFW
- firewalld

### Monitoramento
- Zabbix
- Grafana + Prometheus
- Netdata
- SNMP
