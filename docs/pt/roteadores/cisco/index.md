# Roteadores Cisco

A Cisco é líder global em infraestrutura de rede, com uma ampla linha de roteadores para ISPs, data centers e redes corporativas.

## Modelos Abrangidos

| Série | Uso | Destaques |
|-------|-----|-----------|
| ISR 4000 | Branch / Filiais | IOS-XE, SD-WAN ready, modular |
| ASR 1000 | Edge / Aggregation | QuantumFlow Processor, alta disponibilidade |
| Catalyst 8000 | WAN Edge / SD-WAN | Successor do ASR 1001-X, cloud-ready |

## CLI — Acesso Inicial

```bash
# Acesso padrão via console: 9600 8N1
Router> enable
Router# configure terminal
Router(config)#
```

## Tópicos

- [ISR 4000](./isr-4000) — Branch router, DMVPN, QoS
- [ASR 1000](./asr-1000) — Edge aggregation, BGP, MPLS
- [Catalyst 8000](./catalyst-8000) — SD-WAN, FlexVPN
