# Roteadores

Bem-vindo à seção de roteadores do PacketWiki. Aqui você encontra guias de configuração, referência de comandos e dicas de troubleshooting para roteadores de todos os principais fabricantes.

## Fabricantes

| Fabricante | Modelos Cobertos | Tipo de CLI |
|-----------|-----------------|-------------|
| [Cisco](/pt/roteadores/cisco/) | ISR 4000, ASR 1000, Catalyst 8000 | IOS / IOS-XE |
| [MikroTik](/pt/roteadores/mikrotik/) | hEX, CCR, RB4011 | RouterOS (CLI / Winbox) |
| [Juniper](/pt/roteadores/juniper/) | SRX, MX Series | Junos |
| [Ubiquiti](/pt/roteadores/ubiquiti/) | UniFi Dream Machine, EdgeRouter | EdgeOS / UniFi |
| [Huawei](/pt/roteadores/huawei/) | AR Series, NE Series | VRP |

## Como usar esta seção

Cada página de fabricante contém:

- **Visão geral** — arquitetura geral, principais recursos
- **Configuração inicial** — reset de fábrica, primeiro acesso, config básica
- **Exemplos de configuração** — exemplos reais de CLI/GUI com explicações
- **Problemas comuns** — troubleshooting de problemas conhecidos
- **Referência de comandos** — tabela de referência rápida
